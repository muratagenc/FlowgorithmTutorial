Solving Quadratic Equations

y = ax^2 + bx + c

delta = b^2-4ac

if delta > 0:
	root1 = (-b + sqrt(delta))/2a
	root2 = (-b - sqrt(delta))/2a
if delta = 0:
	root1 = root2 = -b/2a
if delta < 0:
	there is no root

example:
y = 2x^2 - x - 6
2(2^2) - 2 - 6 =? 0
2*4 - 2 - 6 =? 0
